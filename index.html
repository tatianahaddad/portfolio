Pride and Prejudice Quiz App

Find out if you truly know your Pride and Prejudice trivia by submitting your answer when presented with 10 multiple choice questions one at a time, receiving feedback if the answer was correct or not, and receiving your final score results with the option to play again.

HTML was used to load the start page, JavaScript functions were used alongside jQuery libraries to provide interactive material for the user to click and submit answers. In addition, inserting content material such as the questions and multiple choice answers into to the HTML page was accomplished through JavaScript functions and jQuery libraries. CSS was used throughout to create visually striking content and to allow for a seamless UX.
